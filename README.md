A structured 30-days Javascript practice challenge where I consistently build small projects,solved problems and strengthened core JS fundamentals through hands-on coding.

## Overview

This repository contains my daily Javascript practice for 30 days.
Each day focuses on building logic, understanding concepts, and improving problem solving skills.

- Strengthen core JavaScript fundamentals
- Improve problem-solving and logic building
- Practice DOM manipulation and browser APIs
- Build consistency in coding daily
- Learn by creating small mini-projects
